# My Visit.com – Booking.com Clone

A front-end clone of Booking.com's home, login, and register pages, built with **HTML**, **Tailwind CSS**, and **JavaScript**.

## ✨ Features
- Fully responsive home page with search bar and property cards.
- Login and registration modal/pages with form validation.
- Tailwind CSS for rapid, utility-first styling.
- JavaScript for interactivity (e.g., toggle forms, show/hide password).

## 🛠️ Tech Stack
- **HTML5** – Semantic structure
- **Tailwind CSS** – Styling (via CDN or compiled)
- **JavaScript (ES6)** – DOM manipulation, form handling
